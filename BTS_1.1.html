1.Che cos'è Bootstrap?
Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice.
Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, nonché plug-in JavaScript opzionali.
Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi.

2.Cosa devi importare per creare una pagina web con bootstrap?
Si piò includere tramite CDN (Content Delivery Network).
Esempio:
<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
<!-- jQuery library -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js" > </script>
<!-- Popper JS -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js"> &l/script>

3.Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?
La CDN è costituita da un insieme di localizzazioni nel mondo che ridistribuiscono localmente i contenuti dei tuoi server e conservano in cache i file che non richiedono un continuo aggiornamento.
I vantati sono che la maggior parte dei CDN si assicurerà che una volta che un utente richiede un file da esso, verrà servito dal server più vicino a loro, il che porta anche a tempi di caricamento più rapidi.

4.A cosa serve questo attributo del tag meta?
content="width=device-width, initial-scale=1
Imposta la larghezza della pagina per seguire la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo).

5.Che differenza c’è tra Container e Container Fluid? (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)
La .container fornisce un contenitore responsive a larghezza fissa
La .containerfluid fornisce un contenitore a larghezza intera , che copre l'intera larghezza della finestra
